Configuration

Variables

Server configuration is kept in environment variables.

Must change variables:

  1. HTTP_SESSION_SECRET
  2. HTTP_SESSION_SECURE
  3. REST_AUTH_SECRET
  4. ALOWED_HOSTS
  5. ORIGIN

Running application

Before running on prod environment build minified version and set NODE_PATH to include application directory.

Normally it would result in following steps:

  1. Run gulp build
  2. Run NODE_PATH=\".:./node_modules\" node index.js ./config.env
